# Выпускной проект для онлайн курса по NLP.

Тема проекта: "Система семантического поиска и рекомендации новостных статей". В качестве основы я взял датасет из статей, которые я парсил в первом дз:
https://github.com/VladShajdulin/OTUS/blob/main/home_work_1.ipynb.
Это 8 тысяч с сайта Naked Science с новостями из мира науки.

Сначала я реализовал сам семантический поиск. Затем с помощью gpt-модели сгенерировал датасет из поисковых запросов и соответсвующих статей. По нему оценил качество работы поиска и выполнил обучение модели. В конце на основе поиска сделал функцию рекомендаций, которая подбирает близкие по теме статьи.
